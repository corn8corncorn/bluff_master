# 🚀 Bluff Master 本地啟動說明

## 快速啟動（Windows）

### 步驟 1: 準備資料庫

1. 啟動 MySQL 服務
2. 建立資料庫：
```sql
CREATE DATABASE IF NOT EXISTS bluff_master CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 步驟 2: 啟動 Redis

```cmd
redis-server
```

### 步驟 3: 啟動後端

**方法 A - 使用腳本：**
```cmd
cd backend
start.bat
```

**方法 B - 使用命令：**
```cmd
cd backend
mvn spring-boot:run
```

後端將在 `http://localhost:8080` 啟動

### 步驟 4: 啟動前端

開啟**新的命令提示字元視窗**：

**方法 A - 使用腳本：**
```cmd
cd frontend
start.bat
```

**方法 B - 使用命令：**
```cmd
cd frontend
npm install
npm run dev
```

前端將在 `http://localhost:3000` 啟動

## ✅ 驗證

1. 開啟瀏覽器訪問：`http://localhost:3000`
2. 應該看到首頁，可以創建或加入房間

## 📝 注意事項

- **圖片上傳**：本地開發模式使用 Base64 編碼，無需 GCP 配置
- **資料庫**：JPA 會自動建立表格，無需手動執行 SQL
- **Redis**：如果沒有 Redis，可以暫時註解相關配置（但 WebSocket 可能受影響）

## 🔧 常見問題

### MySQL 連接失敗
修改 `backend/src/main/resources/application.yml` 中的資料庫配置：
```yaml
spring:
  datasource:
    username: root  # 改為你的 MySQL 用戶名
    password: your_password  # 改為你的 MySQL 密碼
```

### 端口被占用
- 後端端口：修改 `application.yml` 中的 `server.port`
- 前端端口：修改 `vite.config.js` 中的 `server.port`

### Redis 連接失敗
如果沒有安裝 Redis，可以暫時跳過（但建議安裝以確保完整功能）

# 🚀 完整啟動步驟

## ✅ 步驟 1: 後端已啟動

您已經執行了 `backend/start-jar.bat`，後端服務應該在運行中。

**確認後端是否正常運行：**
- 後端運行在：`http://localhost:8080`
- 查看終端是否有錯誤訊息
- 應該看到類似 "Started BluffMasterApplication" 的訊息

---

## 🎯 步驟 2: 啟動前端服務

### 方式 1: 使用啟動腳本（推薦）

開啟**新的終端視窗**（保持後端運行），執行：

```cmd
cd frontend
start.bat
```

### 方式 2: 手動啟動

```cmd
cd frontend

REM 首次啟動需要安裝依賴
npm install

REM 啟動開發伺服器
npm run dev
```

前端會在 `http://localhost:3000` 啟動。

---

## 🌐 步驟 3: 訪問應用

打開瀏覽器，訪問：
```
http://localhost:3000
```

您應該會看到 Bluff Master 唬爛王的首頁。

---

## 📝 測試流程

1. **創建房間**：點擊「創建房間」
2. **加入房間**：用另一個瀏覽器視窗或裝置加入房間
3. **上傳圖片**：每位玩家上傳 3 張圖片
4. **開始遊戲**：開始進行遊戲

---

## 🆘 常見問題

### 後端啟動失敗？
- 檢查 MySQL 是否運行
- 確認 `application-dev.yml` 中的資料庫配置正確
- 查看終端錯誤訊息

### 前端啟動失敗？
- 確認 Node.js 已安裝：`node -v`
- 如果缺少依賴，執行：`npm install`

### 連接失敗？
- 確認後端在 `http://localhost:8080` 運行
- 確認前端在 `http://localhost:3000` 運行
- 檢查瀏覽器控制台是否有錯誤
